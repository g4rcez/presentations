Entendendo os paranaue

commander

  • Padrão Unix getopt.h
  • Suporte a git style
  • Menu de ajuda automático

chalk

  • Output padronizado entre terminais
  • Adicione estilo ao output
  • Pense como se fosse CSS

signale

  • Diversas features de log
  • Extremamente configurável
  • Error handler

inquirer.js

  • Checkbox
  • Inputs, validações, senhas
  • Listas
  • Confirm
  • Autocomplete (plugin)
  • Integração com Chalk (plugin)
